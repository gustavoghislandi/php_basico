No LMDE, você instala os 3 via apt (separados ou juntos):

    sudo apt update
    sudo apt install apache2 mariadb-server php libapache2-mod-php php-mysql


Isso instala:

    Apache HTTP Server
    MariaDB (alternativa ao MySQL)
    PHP

Depois disso, o servidor já estará rodando.
----------------

NOTA: O MariaDB pode ter desempenho pior quando se usa muito JSON, por exemplo, porque ele guarda JSOn em string, já o MySQL armazena em formato binário interno.

Lembrando: No seu computador (offline) está dividido em cliente e servidor.

Com o EasyPHP tem um lugar específico pra salvar os arquivos, que é em EasyPHP.../data/localweb/, aqui você coloca o arquivos.

No caso de outra versão é dentro da pasta EasyPHP.../www.

No Linux, provavalmente é em /var/www/html/ que é onde está o index.html do servidor apache, por padrão.

--
    Resumo bem enxuto:

        * PHP só funciona em pastas que o servidor web acessa (document root).
        * Windows / EasyPHP: arquivos vão em `localweb` ou `www`.
        * Linux / Apache: padrão é `/var/www/html/`.
        * Fora dessas pastas, o servidor não processa PHP sem configuração extra.
--

Uma supertag: é utilizada para colocar outras linguagens, como PHP.
    Exemplo:

        <?php 
            echo "Olá, Mundo!";
            ?> 

    Tudo que estiver dentro dessas supertags será considerado código PHP.
    Para um código PHP, você pode colocar mais de um comando dentro dela.
    O comando 'echo' é para escrever algo na tela.
    O ponto-vírgula no PHP indica o final do comando.

-------------------
Para enviar os arquivos para o servidor:

    sudo cp url_origem /var/www/html
-------------------

É possível usar tags HTML dentro do comando echo. Exemplo:

    <?php 
        echo "<h2>Olá, Mundo!</h2>";
    ?>

É possível alterar o estilo também, sem problemas.

Na inspeção do código visto no navegador, as supertags são eliminadas.

Isso mostra que o que vai pro cliente é só o HTML, o que o PHP fez foi processar o HTML.

É importante o arquivo estar em UTF-8 e usar extensão .php, senão o que estiver dentro das supertags de PHP não será interpretado.

Perceba que o PHP é uma tecnologia de complemento a HTML e as CSS.
